# Carson Specs

公共的基础约束、工程规范与项目基线。

## 目录

- `app/`: App / Expo / iOS / Android / Web 相关规范
- `backend/`: 后端服务规范
- `infra/`: 部署、发布、运维规范
- `naming/`: 命名规范

## 使用原则

- `specs` 存放规范本身，是权威来源
- 具体项目仓库存放项目事实和项目差异
- `skills` 仓库存放 agent 如何应用这些规范的流程，不替代规范本身
